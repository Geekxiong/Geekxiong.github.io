二、解释

  1. 导包

    tensorflow 是主角
    numpy 是Python的一种开源的数值计算扩展

  2. 创建数据

    这里随机创建了x_data, 而y_data则直接是x_data*0.1 + 0.3, 这里就说明了两者的数值关系。

  3. 创建模型

    Weights和biases会被初始化。
    tf.Variable(initializer,name),参数initializer是初始化参数，name是可自定义的变量名称。
    tf.random_uniform([1], -1.0, 1.0)表示生成一个矩阵里面只有一个数，最小值是-1.0，最大值1.0。
    tf.zeros([1])表示生成一个全零的矩阵，且里面只有一个数。
    最后的 y = Weights * x_data + biases 其实才是模型，与 y_data = x_data*0.1 + 0.3 相对于，计算结果也会是Weights接近0.1，biases接近0.3

  4. 计算误差

    loss 为最终预测的y与真实的y_data的差值。它会起到一个反馈的作用，作用在下一次“学习”中。

  5. 传递误差

    误差传递的工作由optimizer完成，而且这里的传递方法是GradientDescentOptimizer即梯度下降法。

  6. 初始化参数

    运行这句代码才能初始化前面申明的参数

  7. 训练

    首先要获取到TensorFlow的session，然后sess.run(init)进行初始化运行。
    后面的每一次sess.run()都不是单独的，他都属于同一个session，这样就会使得前面的计算对后面的计算有影响，从而纠正预测结果。
