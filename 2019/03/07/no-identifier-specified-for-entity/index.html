No Identifier specified for entity 的解决办法

发表于 2019-03-07 更新于 2019-03-08 分类于 学习 阅读次数：

一、 起因

使用Spring-Data-Jpa， 在定义实体类Entity时，出现了这个异常。

二、 原因

原因有两点

  1. 没有指定主键
    Jpa要求每一个实体都要指定一个主键，所以需要在一个属性上面加上@Id。

  2. 导错包
    这个正是我错误的地方，倒错了包。
    这里的@Id这个注解在两个地方存在

    我们需要的是 javax.persistence.Id，
    而不是org.springframework.data.annotation.Id
